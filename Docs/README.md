# BlackSense Thermo EduKit
IoT-based temperature and heat-transfer monitoring system using ESP32, DS18B20 multi-sensors, MQTT HiveMQ, and Python Dash dashboard.

## Features
- 3 DS18B20 sensors (hot, cold, mixed)
- Real-time dashboard (C, F, K, R)
- Heat transfer calculation (Asas Black)
- Excel auto-export
- Wiring diagrams
- MQTT publishing (HiveMQ)
- Offline/online mode support
